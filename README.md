# Arche & Symbol – Web Design Project

This project is a **final web design project** inspired by the psychology and symbolism
of **Carl Gustav Jung**. The goal is not only to present information, but also to create
an immersive atmosphere through visual design.

---

## Project Purpose

- Present Jung’s concepts such as archetypes, dreams, and symbolism
- Practice HTML and CSS fundamentals
- Create a thematic and visually consistent website
- Improve responsive design skills

---

## Design & Technical Structure

- Dark color palette and gradient transitions were used to reflect the unconscious
  and mystical themes
- Light-colored headings and buttons were applied to maintain contrast
- Classic fonts for headings, readable fonts for content
- Built using **pure HTML & CSS** (no frameworks)
- Layout created with **Flexbox**
- Responsive design achieved using **media queries**
- Hover effects applied to cards, buttons, and navigation elements
